## Flash-envios ##

Se creo una pagina con un chat integrado we capaz de responder preguntas de la empresa ficticia Flash Envios.

Empeze con una arquitectura basica en N8N siguiendo el video de la inmersion, sin embargo tuve que hacer muchas modificaciones para que funiconara, por ejemplo cambie telegram por webhook al igual que agregue un if para que pudiera responder mucho mas rapido, tambien agregue un segundo flujo para preguntas mas generales, despues enlace esto a mi pagina web que cree en visual estudio y que subi atraves de github

N8N, Railway, git, github, visual studio code, adobe, excel, word y cohere.

Ingresar a esta pagina web: https://meal0810.github.io/flash-envios/ despues escribir tus dudas respecto a las politicas de envio y manejo de la empresa ficticia "Flash envios" o tambien buscar usando un tracknumber.
